Project #6: Looking out the Window

Description

The scaffolding on my apartment building just came down, and I’ve been so happy to be able to look out the window again. For one week, I tracked each time I looked out the window and what I saw. I created a visualization of this data in p5.js using arrays, conditionals, and loops.

Process





I logged each viewing in a notebook kept near the window, jotting down the date, time, and a short description. Once I had gathered a few days of data, I started thinking about how it would be visualized. How would it be categorized; how would each unit be represented; do the x and y axies carry particularly meaning. The Dear Data project was immensely helpful in thinking through how to make the data tangible.



The data visualization was made in p5.js using arrays, conditionals, and loops. The color of sky or atmospheric conditions was a recurrent note, and so for my background a I created a gradient sky. Layered on top of these are window-ish boxes, representing each entry.



I ran into some hiccups translating my sketches to code. When I tried to access my text based array, I could only get the word to appear on screen, and I wanted an abstracted object representation. When I tired to do this, it would make a square for each character in the word. Allison Parrish’s Text and type tutorial was immensely helpful in getting me closer to my initial sketch.

Reflection

As I became more conscious of the behavior, I found myself looking for new things, rather than the same thing already recorded. I appreciated that the exercised deepened my looking.

Continuing the project, including photos on the hover would be visually impactful and further impart the action of looking out a window. I’d also like the width of the boxes to be more meaningful. Right now they’re bound to the word length, but taking on a time element would be more meaningful, e.g. How long did I look out the window? What time of day? I’d also like to figure out how to categorize and filter the arrays, and assign colors, so that an additional story of the types of things noticed is more evident.
