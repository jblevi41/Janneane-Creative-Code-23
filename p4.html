Project #4: Experimental Clock

Description

My clock uses an animated spiral of ellipses to compare how a child experiences time to how an adult experiences time. As a mother to a toddler, it's really wonderful to see how we gain our understanding of time. I came across a study that explained why one minute feels like forever when you're young, but flys by as you age. Using a frame rate analogy, younger brains are capturing more frames per minute or second, and adult brains are capturing fewer. So in a sense, a child is experiencing more time per minute, and an adult less. A child is often processing experiences or even everyday phenomona for the first time -- they're building and encoding rich context memories. As adults, we already have a bank of memories stored, so we're able to use more lossy ways of experiencing time, calling forth experiences and learned context.

Process Summary

Using a basic spiral function in p5.js, I generated a formula to alter the animation based on age input. Someone younger than 18, will reveal a denser animation reflecting the higher frame rate. Someone older than 18 will generate a faster more sparse dot pattern.

Process Detail

I began with a series of pseudocode prompts and sketches to understand how I might visualize or code time. Drawing spirals, circles, lines, and waves lent visual metaphors for makring time.

I selected the spiral pattern because I felt it readily references the clock. Time passing, repeating, expanding. There's a lot to explore still regarding the speed, frame rate. It would be interesting to build in a steady gradation of how time is experienced by year. I'd also like to build in a reset funtion after one minute, and/or run the animations side by side.
