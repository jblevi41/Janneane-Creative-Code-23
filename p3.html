Project #3: Parametric Design

Description

I developed a parametric design in p5.js using the random() function along with mousePressed() and some additional experiments with mouseX, mouseY. Through these functions, I made an abstracted interaction of a honeybee pollinating flowers. Run your finger along the trackpad to fly it, and click to make a flower.

Sketch

Process Summary

I started with a sketch to make the flowers grow. The overall process ended up being a patchwork of adaptation - grabbing code from others, changing the shape, color, values. And the result was something I think was even cooler! An abstracted honeybee buzzing around making new flowers.

Process Detail

I realized that Bézier curves would likely be the best way to get a dynamic more asymmetrical flower shape. But they are so tricky! Earlier I had tried Chuck Grimmett’s Vera Molnar exercise to generate random quadrilaterals and really liked it. In a bit of an experiment I decided to sub bezier() for quad() and it worked! This became the structure for randomizing the petals. I still want to do some work on size and placement.

I also tried an alternate version with mouseX, mouseY variables that’s pretty fun!)

I added some code we looked at in class for randomizing color. I attempted to constrain the RGB values to reds/pinks but need to keep working on this.

The bee was an experiment using The Coding Train’s mouseX, mouseY tutorial to make a sort of painting app.

Another feature I’d like to keep working on is a clear or reset so the user can start again.
